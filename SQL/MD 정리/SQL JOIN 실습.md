# SQL JOIN 실습



1. JOIN 을 사용하여 동아리 묶기

```SQL
SELECT A.SNO, A.NAME, A.YEAR, A.CLUB_NO, B.CLUB_NM, B.PLC
FROM STUDENT A 
INNER JOIN MAJOR_CLUB B ON B.YEAR = A.YEAR 
	AND B.CLUB_NO = B.CLUB_NO
WHERE A.SYEAR = 1  ;
```



2. 

```SQL
SELECT A.SNO '학번', A.NAME '이름', A.YEAR '년도', B.CLUB_NM '동아리 명', C.NAME '동아리장'
	FROM STUDENT A
      JOIN MAJOR_CLUB B 
         ON B.YEAR = A.YEAR 
       AND B.CLUB_NO = A.CLUB_NO
	  JOIN STUDENT C 
         ON B.CLUBHD = C.STUD_NO
  WHERE A.SYEAR = 1;
```



3. 

```SQL
SELECT A.SNO '학번', A.NAME '이름', B.CLUB_NM '동아리 명', C.NAME '동아리장', D.NAME '지도교사'
	FROM STUDENT A
      JOIN MAJOR_CLUB B ON B.YEAR = A.YEAR 
       AND B.CLUB_NO = A.CLUB_NO
	  JOIN STUDENT C ON B.CLUBHD = C.STUD_NO
	  JOIN TEACHER D ON D.TEMP_NO = B.CHRG_TEAC
  WHERE D.NAME = '엄영석'
ORDER BY B.CLUB_NM, A.SNO;	
```



4. 

```SQL
SELECT D.NAME '지도교사' , B.CLUB_NM '동아리명', C.NAME '동아리장', COUNT(*) '총 인원수'
    FROM STUDENT A
    JOIN MAJOR_CLUB B ON A.YEAR = B.YEAR
    AND A.CLUB_NO = B.CLUB_NO
    JOIN STUDENT C ON B.CLUBHD = C.STUD_NO
    JOIN TEACHER D ON D.TEMP_NO = B.CHRG_TEAC
WHERE D.NAME = '서무성'
  GROUP BY  B.CLUB_NM-- , C.NAME, D.NAME
  ORDER BY B.CLUB_NM, A.SNO;
```



5. 

```SQL
SELECT D.NAME '지도교사', B.CLUB_NM '동아리명', A.SYEAR '학년', COUNT(*) '총인원수'
    FROM STUDENT A
      JOIN MAJOR_CLUB B ON A.YEAR = B.YEAR
       AND A.CLUB_NO = B.CLUB_NO
	  JOIN STUDENT C ON C.STUD_NO = B.CLUBHD
      JOIN TEACHER D ON D.TEMP_NO = B.CHRG_TEAC
      WHERE D.NAME = '김학동'
      GROUP BY D.NAME, B.CLUB_NM, C.NAME, A.SYEAR
      ORDER BY B.CLUB_NM, A.SNO;
```
